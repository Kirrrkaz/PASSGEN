# PASSGEN
This script generates all characters of the first password segment keeping the tail 


## Example :

<h> Source password: 123QQ
<br> Need generate all variables with keeping the tail QQ </br>
</h>

<ol> Input:

<li> pass: 123 </li>
<li> tail-pass: QQ </li>
</ol>

<ol> Output:

<li> 123QQ </li>
<li> 132QQ </li>
<li> 213QQ </li>
<li> 231QQ </li>
<li> 312QQ </li>
<li> 321QQ </li>

</ol>
