# PASSGEN
This script generates all variables of the first password segment keeping the tail 


## Example :

<h> Source password: 123QQ
Need generate all variables with keeping the tail QQ 
</h>

<ol> Input:

<li> pass: 123 </li>
<li> tail-pass: QQ </li>
</ol>

<ol> Output:

<li> 123QQ </li>
<li> 132QQ </li>
<li> 213QQ </li>
<li> 231QQ </li>
<li> 312QQ </li>
<li> 321QQ </li>

</ol>
